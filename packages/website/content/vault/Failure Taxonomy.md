# Failure Taxonomy

Classify harness failures into stable buckets:

- planning error
- tool execution error
- permission error
- validation miss
- reporting mismatch

A shared taxonomy improves triage and alert routing.
It also reveals where guardrails or tests are missing.

Use taxonomy labels in [[trace-and-replay]] events.
Use them to balance your [[evaluation-corpus-design]] dataset.
